# 快速开始指南

## ⚡ 5分钟上手

### 步骤 1: 准备模型文件

将你的 TensorFlow Lite 模型文件复制到：
```bash
app/src/main/assets/digit_recognition_model.tflite
```

### 步骤 2: 启动 API 服务（可选）

如果要使用在线模型，在终端运行你的 Python API：
```bash
# 示例
python api_server.py
# 确保服务运行在 localhost:8000
```

### 步骤 3: 运行应用

在 Android Studio 中：
1. 打开项目
2. 等待 Gradle 同步完成
3. 点击 Run 按钮 ▶️
4. 选择模拟器或连接的设备

或使用命令行：
```bash
./gradlew installDebug
```

### 步骤 4: 开始识别

1. **手写模式**:
   - 选择 "手写" 输入方式
   - 在白色画布上用手指绘制数字
   - 点击 "识别" 按钮
   
2. **拍照模式**:
   - 选择 "拍照/相册" 输入方式
   - 点击 "拍照" 或 "相册"
   - 选择或拍摄包含数字的图片
   - 点击 "识别" 按钮

### 步骤 5: 查看结果

应用会显示：
- 🎯 识别的数字
- 📊 置信度百分比
- 📈 所有数字的概率分布

## 🔍 测试技巧

### 手写识别最佳实践
- ✅ 在画布中央绘制
- ✅ 保持数字适中大小
- ✅ 笔画连贯清晰
- ❌ 避免过小或过大
- ❌ 避免多次重叠涂抹

### 拍照识别最佳实践
- ✅ 确保光线充足
- ✅ 数字清晰可见
- ✅ 背景简单干净
- ❌ 避免模糊或过暗
- ❌ 避免复杂背景

## 🐛 快速故障排除

| 问题 | 解决方案 |
|------|----------|
| 找不到模型文件 | 确认 `.tflite` 文件在 `assets` 目录 |
| API 连接失败 | 检查 API 服务是否运行在 8000 端口 |
| 相机权限被拒绝 | 在设置中手动授予相机权限 |
| 识别不准确 | 尝试重新绘制，确保笔迹清晰 |

## 📱 支持的 Android 版本

- 最低版本: Android 7.0 (API 24)
- 目标版本: Android 14 (API 36)
